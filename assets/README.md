# Assets de HQS Radio

Esta carpeta debe contener los siguientes archivos:

## Archivos Requeridos

- `icon.png` - Icono principal de la app (1024x1024px)
- `splash.png` - Pantalla de inicio (2048x2048px)
- `adaptive-icon.png` - Icono adaptativo para Android (1024x1024px)
- `favicon.png` - Favicon para web (48x48px)

## Instrucciones

1. Crea las imágenes con las dimensiones especificadas
2. Usa el color azul (#1E3A8A o #3B82F6) como color principal
3. Incluye el logo o texto "HQS Radio"
4. Asegúrate de que el fondo del splash screen sea azul (#1E3A8A)

## Herramientas Recomendadas

- Figma (diseño)
- Canva (diseño rápido)
- AppIcon.co (generación automática)
- Icon Kitchen (generación de iconos)
