# Changelog

Todos los cambios notables en este proyecto serán documentados en este archivo.

El formato está basado en [Keep a Changelog](https://keepachangelog.com/es-ES/1.0.0/),
y este proyecto adhiere a [Semantic Versioning](https://semver.org/lang/es/).

## [1.0.0] - 2024-01-XX

### Agregado
- Reproductor de radio IceCast 2 funcional
- Panel de administración para gestionar múltiples streams
- Controles avanzados de audio (play/pause, volumen)
- Guardado de streams favoritos
- Reconexión automática en caso de pérdida de conexión
- Interfaz moderna con tema azul
- Reproducción en segundo plano
- Configuración persistente (volumen, stream actual)
- Scripts de utilidad para verificación y generación de assets
- Documentación completa (README, guía de publicación)
- Configuración lista para publicación en App Store y Google Play

### Características Técnicas
- React Native con Expo
- Expo AV para reproducción de audio
- AsyncStorage para persistencia
- Componentes modulares y reutilizables
- Utilidades centralizadas (storage, validation, audio)
- Hook personalizado para manejo de audio
- Configuración centralizada

### Documentación
- README.md con instrucciones completas
- PUBLICACION.md con guía de publicación
- CONTRIBUTING.md con guía de contribución
- Instrucciones detalladas para creación de assets
- Scripts de verificación y setup
